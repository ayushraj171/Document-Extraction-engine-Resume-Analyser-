# Document Extraction Engine & Resume Analyzer

An AI-powered web application that extracts and analyzes information from resumes and documents using the Gemini API.

## Features
- Resume/document upload
- AI-based text extraction
- Resume analysis
- Structured output generation

## Tech Stack
- Python
- Gemini API
- Streamlit / Flask
- HTML/CSS

## How to Run

1. Install dependencies

```bash
pip install -r requirement.txt

```

2. Create a `.env` file

```env
GEMINI_API_KEY=your_api_key_here
```

3. Run the project

```bash
python app.py
```
